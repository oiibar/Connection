<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the project</p>

```
git clone https://github.com/oiibar/Connection.git
```

<p>2. Navigate to the server folder</p>

```
cd Cat_Wiki/server
```

<p>3. Install server-side packages</p>

```
npm i
```

<p>4. Run the server</p>

```
npm start
```
